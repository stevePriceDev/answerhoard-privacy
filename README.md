# answerhoard-privacy

This repository hosts the AnswerHoard privacy policy content and a simple static privacy page.

Files included:

- `privacy.md` — the privacy policy source written in Markdown.
- `index.html` — a lightweight HTML version of the privacy policy for easy publishing.

The page is designed to describe how the AnswerHoard browser extension stores data locally, handles exports, and avoids analytics or remote telemetry.
