AnswerHoard Privacy Policy

Last updated: 2026-05-14

AnswerHoard is a browser extension that helps you keep a local copy of information you enter on job application forms.

What data the extension collects

When you use AnswerHoard on a web page, it may capture:

  • Field snapshots — labels, questions, and the values you typed or selected in visible form controls (for example text fields, text areas, selects, checkboxes, and radio buttons).
  • File uploads (names only) — if you attach a file, only the file name is stored, not the file contents.
  • Page context — the page URL, document title, and any job title or company name the extension can infer from the page.

The extension does not intentionally capture:

  • Password fields (type="password") or common anti-bot tokens (for example reCAPTCHA / Turnstile response fields).
  • Hidden form fields used for site mechanics.

Where your data is stored

All captured data is stored only on your device using the browser’s extension storage (storage.local). It is not sent to AnswerHoard’s servers — the extension does not operate a backend service for your answers.

Export

If you use Export to JSON in the extension’s settings page, a file is downloaded to your computer. That file contains the same data held in extension storage. You are responsible for how you store or share that file.

Analytics and third parties

AnswerHoard does not include analytics, advertising, or remote telemetry. It does not sell your data.

Retention and deletion

Data remains in extension storage until you remove it (for example by deleting individual saved applications or clearing extension data in the browser). Uninstalling the extension may remove stored data depending on your browser’s behavior.
